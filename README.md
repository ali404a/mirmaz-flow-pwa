# mirmaz FLOW — نظام إدارة الإنتاج التسويقي والإبداعي

نظام متكامل لإدارة سير عمل المحتوى لشركة **mirmaz** (براندَي mirmaz Academy و mirmaz Zone) — من إنشاء المهمة، مروراً بكتابة الكابشن والتصميم والمراجعة والموافقة، وصولاً للنشر المجدول. واجهة عربية كاملة (RTL) مع نظام صلاحiات لكل تخصص، دردشة، تقويم نشر، وتقارير.

## وضعان للتشغيل

### الوضع 1: تجريبي (يعمل فوراً — بدون إعداد)

افتح `frontend/index.html` بدبل كليك. التطبيق يعمل كاملاً ويحفظ البيانات في المتصفح (LocalStorage). مثالي للعرض والتجربة.

> ملاحظة: في هذا الوضع البيانات محلية على جهازك فقط، ولا يوجد تسجيل دخول حقيقي.

### الوضع 2: موقع حقيقي (مع قاعدة بيانات وتسجيل دخول)

اتبع **`docs/دليل-الرفع.md`** خطوة بخطوة. باختصار:

1. أنشئ مشروع **Supabase** (مجاني)
2. شغّل ملفات `database/` الثلاثة بالترتيب
3. أنشئ حساب السوبر أدمن
4. انشر `backend/create-user-function.ts`
5. اربط المفاتيح في `frontend/supabase-client.js`
6. ارفع على **Vercel**

النتيجة: موقع فعلي، كل شخص يدخل بحسابه ويرى ما يخصّه فقط.

## الميزات

**سير العمل (الخرائط):**
- كل مهمة لها خريطة دوائر تفاعلية: كتابة الكابشن ← تكست دزاين ← التصميم ← المراجعة ← الموافقة ← النشر
- المرحلة الجارية ملوّنة ونابضة، المنجزة خضراء، القادمة باهتة
- زر تعديل لكل مرحلة مع **سجل من عدّل ومتى**
- أي تعديل يعيد المسار **للتدقيق ثم موافقة الإدارة** تلقائياً

**النشر المنظّم:**
- يُفتح فقط بعد موافقة الإدارة
- يجمع الكابشن النهائي + ملف الدرايف (من المصمم) في مكان واحد
- تحديد يوم وساعة النشر
- أزرار: تأجيل / جاري النشر / تم النشر

**التقويم التفاعلي:**
- تنقّل بين الشهور، عرض منشورات كل يوم
- المنشور المكتمل يظهر بالأخضر

**المهام الشخصية:** مساحة معزولة لكل موظف (يراها هو فقط)

**الدردشة:** واجهة شبيهة بتيمز، مهيّأة للربط بـ Microsoft Teams (مشفّر)

**أخرى:** تقارير وتحليلات، صفحة الفريق، إشعارات، بحث، مشاركة المهام، تقويم

## الأدوار والصلاحiات (9 تخصصات)

| التخصص | الصلاحية |
|--------|----------|
| سوبر أدمن | كل شيء + إنشاء الحسابات والأقسام |
| مانيجر | إدارة الموظفين/الأقسام + رؤية الكل |
| كوردنيتر | إنشاء المهام وبناء الخرائط |
| الإدارة | موافقة/رفض المهام |
| مدقق | مراجعة المراحل |
| كاتب كابشن / مصمم / تكست دزاينر / مونتير | العمل على مراحلهم فقط |

الصلاحiات مفروضة على **3 طبقات**: الواجهة + قاعدة البيانات (RLS) + triggers الخادم.
